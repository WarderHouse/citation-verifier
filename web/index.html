Citation Verifier

Paste your reference list and check which references actually exist, before you cite them.

Each reference's title and DOI are looked up in two public databases (CrossRef and OpenAlex). No private data, no AI, no account. The look-up happens from your browser.

One reference per line, in whatever style you have (APA works well). The checker shows the title it read from each line so you can see what it looked up.

"Verified" means two databases agree the work exists, "check this" means only one did, and "not found" means neither did, which usually points to a fabricated or mistyped reference (though a very obscure work can also miss). It confirms a reference is real; it does not confirm the reference says what you claim it says. Reading each pasted line is best-effort, so glance at the title it pulled out.
